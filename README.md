# Password-Based Circuit Breaker System

## Overview

This project is an Arduino-based Password Protected Circuit Breaker System designed to improve electrical safety by preventing unauthorized switching of electrical loads.

Only users with the correct password can operate the connected electrical load. Incorrect password attempts activate a buzzer to indicate unauthorized access.

---

## Features

- Password Protected Switching
- LCD Display Interface
- Keypad Input
- Relay-Based Load Control
- Buzzer Alarm for Wrong Password
- Safe Electrical Operation

---

## Components Used

- Arduino Uno
- 4x4 Matrix Keypad
- 16x2 LCD Display
- Relay Module
- Miniature Circuit Breaker (MCB)
- Buzzer
- Power Supply
- Connecting Wires

---

## Working Principle

1. User enters password using the keypad.
2. Arduino verifies the password.
3. If the password is correct:
   - Relay turns ON.
   - Electrical load is energized.
4. If the password is incorrect:
   - Relay remains OFF.
   - Buzzer sounds.
   - LCD displays an error message.
